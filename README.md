# Mender
### Mender is a movie recommender system built on Python [Streamlit](https://streamlit.io/) Library
### Uses trained ML models to actually precdict movies according to your interests.
#### Some ML algorithms used are:
####  > Count Vectorizer
####  > Consine Similarity
